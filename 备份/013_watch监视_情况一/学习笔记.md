# 013 watch 监视：情况一

## 本节目标

掌握 `watch` 的基本用法：监视 ref 定义的**基本类型**数据。

## 核心知识点

### 1. watch 的基本语法

```ts
import { ref, watch } from 'vue'

let sum = ref(0)

const stopWatch = watch(sum, (newValue, oldValue) => {
  console.log('sum变化了', newValue, oldValue)
})
```

| 参数 | 作用 |
|------|------|
| 第一个参数 | 被监视的数据（这里是 ref 对象 `sum`） |
| 第二个参数 | 回调函数，收到 `(newValue, oldValue)` |
| 第三个参数（可选） | 配置对象 `{ deep, immediate }`（后续课程会用到） |
| 返回值 | 一个停止函数，调用后停止监视 |

### 2. newValue 和 oldValue

```ts
watch(sum, (newValue, oldValue) => {
  console.log(newValue, oldValue)
  // 点按钮 sum 从 0 变成 1 → newValue=1, oldValue=0
})
```

### 3. 停止监视

```ts
const stopWatch = watch(sum, (newValue, oldValue) => {
  if (newValue >= 10) {
    stopWatch()  // sum 达到 10 后停止监视
  }
})
```

watch 返回一个函数，调用它就可以取消监视。适用于"只需要监听一次"或"条件满足后停止"的场景。

## 页面行为

- 每次点"点我sum+1"，控制台打印新值和旧值
- sum 达到 10 后，不再打印（监视已停止）
