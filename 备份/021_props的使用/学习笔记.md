# 021 props 的使用

## 本节目标

掌握 Vue 3 中父组件通过 props 向子组件传数据，结合 TS 做类型约束。

## 核心知识点

### 1. defineProps 的三种写法

**写法一：只接收（不限制类型）**
```ts
defineProps(['list'])
```

**写法二：接收 + 类型约束**
```ts
defineProps<{ list: Persons }>()
```

**写法三：接收 + 类型 + 可选性 + 默认值**
```ts
withDefaults(defineProps<{ list?: Persons }>(), {
  list: () => [{ id: '01', name: '康师傅·王麻子·特仑苏', age: 19 }]
})
```

### 2. 父组件传数据

```ts
// App.vue
let personList = reactive<Persons>([
  { id: '01', name: '张三', age: 18 },
  { id: '02', name: '李四', age: 20 },
  { id: '03', name: '王五', age: 22 }
])
```
```html
<!-- 父组件通过属性传数据 -->
<Person :list="personList" />
<!-- 或不传，让子组件用默认值 -->
<Person />
```

### 3. withDefaults 的作用

```ts
withDefaults(defineProps<{ list?: Persons }>(), {
  list: () => [{ id: '01', name: '康师傅', age: 19 }]
})
```

- `list?: Persons` 中的 `?` 表示 list 是**可选的**（父组件可以不传）
- `withDefaults` 第二个参数指定默认值
- **注意：默认值如果是数组/对象，必须用箭头函数返回**（`() => [...]`）

### 4. 类型来自 020

```ts
// types/index.ts（020 定义的）
export interface PersonInter { id: string; name: string; age: number }
export type Persons = PersonInter[]
```

021 直接复用了这些类型来约束 props。

### 5. props 是只读的

子组件拿到的 props 是**只读**的，不能直接修改。需要修改时应该通过 emit 通知父组件（后续课程会讲）。

## defineProps 速查

| 写法 | 类型约束 | 默认值 | 可选 |
|------|----------|--------|------|
| `defineProps(['list'])` | ❌ | ❌ | ❌ |
| `defineProps<{list: Persons}>()` | ✅ | ❌ | ❌ |
| `withDefaults(defineProps<{list?: Persons}>(), {...})` | ✅ | ✅ | ✅ |
