# 001 写一个 App 组件

## 本节目标

从零创建一个最小的 Vue 3 应用：一个根组件 `App.vue` + 一个入口文件 `main.ts`。

## 核心知识点

### 1. Vue 单文件组件（SFC）的三段式结构

`App.vue` 由三部分组成：

| 区块 | 作用 | 本节代码 |
|------|------|----------|
| `<template>` | 写 HTML 结构 | 一个 `<h1>你好啊！</h1>` |
| `<script lang="ts">` | 写组件逻辑（TS） | `export default { name: 'App' }` |
| `<style>` | 写样式 | `.app` 盒子样式 |

### 2. `name` 属性

```ts
export default {
  name: 'App' // 组件名，方便调试时在 DevTools 中识别
}
```

### 3. 入口文件 main.ts 做了两件事

```ts
import { createApp } from 'vue'  // ① 引入创建应用的工厂函数
import App from './App.vue'      // ② 引入根组件
createApp(App).mount('#app')     // ③ 创建实例 → 挂载到 index.html 中 id="app" 的 DOM
```

> `createApp(App)` 创建一个 Vue 应用实例，`.mount('#app')` 把它挂到页面上。

## 关键代码解读

**App.vue** — 用的还是 Options API（`export default {}`），但此时还没有 data、methods，只有一个 `name`。

**main.ts** — 整个课程所有版本的 main.ts 都是这个骨架，后续会在此基础上加 router、pinia 等。

## 与后续课程的关系

这是地基。002 会在 App 里引入子组件；003 开始用 setup 函数；006 之后切换到 `<script setup>` 语法糖。但 **createApp + mount 这行代码始终不变**。
