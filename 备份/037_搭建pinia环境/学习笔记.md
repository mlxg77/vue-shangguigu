# 037 搭建 Pinia 环境

## 本节目标

把 Pinia 安装到 Vue 应用中——纯环境搭建，不涉及业务逻辑。

## 相对 036 的变化

只有 `main.ts` 变了，组件代码完全不变：

```ts
import { createApp } from 'vue'
import App from './App.vue'
import { createPinia } from 'pinia'  // ① 引入

const app = createApp(App)
const pinia = createPinia()  // ② 创建
app.use(pinia)               // ③ 安装
app.mount('#app')
```

## 核心知识点

### 安装 Pinia 的三步

| 步骤 | 代码 | 说明 |
|------|------|------|
| 引入 | `import { createPinia } from 'pinia'` | 从 pinia 包导入工厂函数 |
| 创建 | `const pinia = createPinia()` | 创建 pinia 实例 |
| 安装 | `app.use(pinia)` | 挂到 Vue 应用上 |

### 前置条件

需要先安装 pinia 包：
```bash
npm i pinia
```

### 和 router 的安装方式一样

```ts
app.use(router)  // 安装路由（024）
app.use(pinia)   // 安装 Pinia（037）
```

> Vue 插件的安装方式都是 `app.use(xxx)`。

## 037 没干什么

- 没有创建任何 store
- 没有搬数据
- 组件行为和 036 **完全一样**
- 只是让后续的 store 有地方可以注册
