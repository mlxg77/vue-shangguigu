# 019 标签的 ref 属性

## 本节目标

用 `ref` 属性给 DOM 元素起名字，在 JS 中获取 DOM 引用；了解 `defineExpose` 暴露数据给父组件。

## 核心知识点

### 1. 用 ref 属性标记 DOM 元素

```html
<h2 ref="title2">北京</h2>
```

`ref="title2"` 是给这个 `<h2>` 元素起了一个名字叫 `title2`。

### 2. 在 JS 中获取 DOM 引用

```ts
import { ref } from 'vue'

let title2 = ref()  // 声明一个同名的 ref 变量

function showLog() {
  console.log(title2.value)  // 输出 <h2>北京</h2> 这个 DOM 元素
}
```

**关键步骤：**
1. 模板中：`ref="title2"`（给元素起名）
2. JS 中：`let title2 = ref()`（声明同名变量）
3. 使用：`title2.value` 就是那个 DOM 元素

> 变量名必须和 ref 属性值一致，Vue 会自动把 DOM 元素挂到对应的 ref 变量上。

### 3. ref 的两种用途对比

| 用法 | 含义 | value 是什么 |
|------|------|-------------|
| `let count = ref(0)` | 创建响应式数据 | 数字 0 |
| `let title2 = ref()` + `<h2 ref="title2">` | 获取 DOM 引用 | DOM 元素 `<h2>` |

> 同一个 `ref()` 函数，在不同场景下用途不同：一个包数据，一个接 DOM。

### 4. defineExpose —— 暴露数据给父组件

```ts
let a = ref(0)
let b = ref(1)
let c = ref(2)

defineExpose({ a, b, c })
```

`<script setup>` 默认是封闭的，组件内部的数据**父组件访问不到**。`defineExpose` 把指定的数据/方法暴露出去，让父组件能通过 `ref` 访问子组件的数据。

> 这是组件通信的一种方式，044 会更详细地讲。
