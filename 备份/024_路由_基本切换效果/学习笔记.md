# 024 路由：基本切换效果

## 本节目标

搭建 Vue Router 基础环境，实现页面切换效果（首页/新闻/关于）。

## 核心知识点

### 1. 路由三件套

| 文件 | 作用 |
|------|------|
| `router/index.ts` | 创建路由器，配置路由规则 |
| `main.ts` | 把路由器挂到 app 上 |
| `App.vue` | 用 `<RouterLink>` 导航 + `<RouterView>` 展示 |

### 2. 创建路由器（router/index.ts）

```ts
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../components/Home.vue'
import News from '../components/News.vue'
import About from '../components/About.vue'

const router = createRouter({
  history: createWebHistory(),  // 工作模式（026 会讲）
  routes: [
    { path: '/home', component: Home },
    { path: '/news', component: News },
    { path: '/about', component: About },
  ]
})

export default router
```

**routes 配置项：**
- `path`：URL 路径
- `component`：匹配到该路径时渲染的组件

### 3. 挂载路由器（main.ts）

```ts
import router from './router'
const app = createApp(App)
app.use(router)  // 安装路由
app.mount('#app')
```

### 4. 导航与展示（App.vue）

```html
<!-- 导航区：RouterLink 切换路径 -->
<RouterLink to="/home" active-class="xiaozhupeiqi">首页</RouterLink>
<RouterLink to="/news" active-class="xiaozhupeiqi">新闻</RouterLink>
<RouterLink to="/about" active-class="xiaozhupeiqi">关于</RouterLink>

<!-- 展示区：RouterView 根据当前 URL 渲染对应组件 -->
<RouterView />
```

| 组件 | 作用 |
|------|------|
| `<RouterLink to="/home">` | 渲染成 `<a>` 标签，点击切换 URL（不刷新页面） |
| `<RouterView>` | 占位符，当前 URL 匹配到的组件在这里渲染 |
| `active-class="xxx"` | 当前激活的 RouterLink 自动加上这个 class |

### 5. 路由工作原理

```
点击 RouterLink → URL 变成 /news → 路由器匹配 routes → 找到 News 组件 → 在 RouterView 中渲染
```

整个过程**不刷新页面**（SPA 单页应用），由 Vue Router 在前端完成。
