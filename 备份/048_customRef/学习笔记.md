# 048 customRef

## 本节目标

掌握 `customRef`——自定义 ref 的依赖收集和触发逻辑，实现**防抖**等高级效果。

## 核心知识点

### 1. customRef 的语法

```ts
import { customRef } from 'vue'

let myRef = customRef((track, trigger) => {
  return {
    get() {
      track()        // 告诉 Vue：这个值需要被追踪
      return value   // 返回值
    },
    set(newValue) {
      value = newValue
      trigger()      // 告诉 Vue：值变了，请更新视图
    }
  }
})
```

| 函数 | 何时调用 | 作用 |
|------|---------|------|
| `get()` | 数据被读取时 | 必须调用 `track()`，返回当前值 |
| `set()` | 数据被修改时 | 必须调用 `trigger()`，更新值 |

### 2. 本节实现：防抖 ref

`useMsgRef.ts` 实现了一个带延迟效果的 ref——输入后延迟 2 秒才更新视图：

```ts
import { customRef } from 'vue'

export default function(initValue: string, delay: number) {
  let timer: number

  let msg = customRef((track, trigger) => {
    return {
      get() {
        track()              // 追踪依赖
        return initValue     // 返回当前值
      },
      set(value) {
        clearTimeout(timer)  // 清除上一次的定时器（防抖）
        timer = setTimeout(() => {
          initValue = value  // 更新值
          trigger()          // 触发视图更新
        }, delay)            // 延迟 delay 毫秒后才真正更新
      }
    }
  })

  return { msg }
}
```

### 3. 组件中使用

```ts
// App.vue
import useMsgRef from './useMsgRef'

// 使用 Vue 提供的默认 ref：数据一变，页面就更新
// let msg = ref('你好')

// 使用 useMsgRef：数据变化后延迟 2000ms 才更新页面
let { msg } = useMsgRef('你好', 2000)
```

```html
<h2>{{ msg }}</h2>
<input type="text" v-model="msg">
```

> 在输入框中快速输入时，`msg` 的值立即改变，但页面上的 `{{ msg }}` 会延迟 2 秒才更新。
> 每次输入都会清除上一次的定时器，只有最后一次输入后 2 秒才真正更新（防抖效果）。

### 4. customRef vs ref

| | ref | customRef |
|--|-----|-----------|
| 依赖收集 | 自动 | 手动调用 `track()` |
| 触发更新 | 自动 | 手动调用 `trigger()` |
| 自定义逻辑 | ❌ | ✅（如防抖、节流、验证等） |
| 使用复杂度 | 简单 | 较高 |

### 5. customRef 的工作原理

```
读取 msg        →  get() 调用  →  track()  →  Vue 记录依赖
修改 msg        →  set() 调用  →  (延迟)    →  trigger()  →  Vue 更新视图
```

- `track()`：相当于"订阅"——告诉 Vue 这个数据被谁使用了
- `trigger()`：相当于"发布"——通知 Vue 数据变了，需要更新

### 6. 与 023（自定义 hooks）的联系

`useMsgRef` 本质上就是一个**自定义 hook**——把 `customRef` 逻辑封装成可复用的函数，和 023 的 `useSum`、`useDog` 思想完全一致。
