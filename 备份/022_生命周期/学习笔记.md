# 022 生命周期

## 本节目标

掌握 Vue 3 Composition API 的生命周期钩子函数。

## 核心知识点

### 1. Vue 3 的四个阶段 + 六个钩子

```
创建 → 挂载 → 更新 → 卸载
```

| 阶段 | 钩子函数 | 时机 |
|------|----------|------|
| 创建 | （setup 本身） | setup 执行时就是创建阶段 |
| 挂载前 | `onBeforeMount` | 组件挂载到 DOM 之前 |
| 挂载完毕 | `onMounted` | 组件已挂载到 DOM |
| 更新前 | `onBeforeUpdate` | 响应式数据变化、DOM 更新前 |
| 更新完毕 | `onUpdated` | DOM 更新完成 |
| 卸载前 | `onBeforeUnmount` | 组件即将被销毁 |
| 卸载完毕 | `onUnmounted` | 组件已销毁 |

### 2. 代码写法

```ts
import { onBeforeMount, onMounted, onBeforeUpdate, onUpdated, onBeforeUnmount, onUnmounted } from 'vue'

// 创建（不需要钩子，setup 本身就是）
console.log('创建')

onBeforeMount(() => { console.log('挂载前') })
onMounted(() => { console.log('挂载完毕') })
onBeforeUpdate(() => { console.log('更新前') })
onUpdated(() => { console.log('更新完毕') })
onBeforeUnmount(() => { console.log('卸载前') })
onUnmounted(() => { console.log('卸载完毕') })
```

### 3. Vue 2 vs Vue 3 生命周期对比

| Vue 2 | Vue 3 (Composition API) |
|-------|------------------------|
| beforeCreate | setup() |
| created | setup() |
| beforeMount | onBeforeMount |
| mounted | onMounted |
| beforeUpdate | onBeforeUpdate |
| updated | onUpdated |
| beforeDestroy | onBeforeUnmount |
| destroyed | onUnmounted |

> Vue 2 的 beforeCreate 和 created 合并到了 setup 中——setup 本身就在这两个阶段之间执行。

### 4. 常用场景

| 钩子 | 典型用途 |
|------|----------|
| `onMounted` | 发请求、初始化第三方库（DOM 已就绪） |
| `onBeforeUnmount` | 清理定时器、取消订阅、释放资源 |
| `onUpdated` | 数据更新后操作 DOM |

### 5. 控制台输出顺序

首次挂载时：
```
创建 → 挂载前 → 子组件挂载完毕 → 父组件挂载完毕
```

点击按钮更新时：
```
更新前 → 更新完毕
```
